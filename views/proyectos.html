Nuestro portafolio

En Latasoft desarrollamos múltiples proyectos los cuales abarcan el diseño web y el desarrollo de aplicaciones móviles. Estos proyectos incluyen la creación de sitios web interactivos, aplicaciones móviles innovadoras y soluciones digitales centradas en mejorar la experiencia del usuario.

Soap Ayuda

Mediante el Plan Latasoft se trabajó una página estructurada tipo landing page para recaudación de seguros SOAP para ONG.
Algunas marcas adheridas a este proyecto fueron Mutual de seguro junto con seguros obligatorios para automóviles

Wok Panda

Mediante el Plan Latasoft nos centramos en la creación y expansión de la página web para el tenedor libre más grande de Chile. Como parte de nuestro compromiso continuo con la excelencia en el desarrollo web y la satisfacción del cliente expandimos la presencia en línea de este establecimiento líder en su categoría.

Sur Model

Mediante el Plan Maestro se trabajó en la implementación de una plataforma de gestión documental, solicitud y posterior toma de promoción de eventos para sus promotoras.
Adicional a esto se trabajó en una extensión de aplicación móvil para la ofertas de trabajos temporales

Autodraft

Mediante el Plan Maestro trabajamos en una página web estilo e-commerce, la cual contiene una plataforma interna con gestión de inventario y gestión documental interna, adicional a esto, la plataforma cuenta con un sistema de pagos para realizar las compras correspondientes.

DesarrollaT

En Latasoft desarrollamos una e-commerce destinada a la empresa DesarrollaT, implementamos el diseño y posterior programación de este sistema.
En este proyecto desarrollamos cada una de las vistas, las cuales incluyen su respectiva sección de pago, contacto, etc.

Cepcon

Para la plataforma desarrollada a Cepcon, implementamos la programación de la web según los requerimientos solicitados por la empresa, desarrollo web el cual se realizó con vistas para los distintos dispositivos.

Fuerza de ventas

Para Fuerza de ventas desarrollamos una plataforma de servicios y contacto.
En Latasoft desarrollamos el diseño visual y posterior programación de este sistema.

Total Medios

Con Total Medios desarrollamos la programación web con sus distintas secciones disponibles.
Para este proyecto integramos distintas funcionalidades web solicitadas, teniendo así como resultado final una página atractiva y funcional.

ProyectaCorp

Para ProyectaCorp se realizó una estructura de sistema de software interno a la empresa, añadiendo la funcionalidad correspondiente y el sistema de seguridad necesario para su correcto funcionamiento interno.

DuocUC

Para DuocUC se realizó un programa de realidad virtual y realidad aumentada, relacionada a la escuela de construcción, especificamente a la sede de Valparaiso.

Masisa Venezuela

Para el desarrollo de Masisa Venezuela realizamos la landing page con sus distintas funcionalidades y su respectivo diseño representativo.

Publicity Ads

En Publicity Ads trabajamos con el desarrollo de una landing page orientada a publicidad en internet y presencia radial.

Algunas de las Apps que realizamos

Surmodel

Snack App
